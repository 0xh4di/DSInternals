# Remove-ADDBObject

## SYNOPSIS
Physically removes specified object from a ntds.dit file, making it semantically inconsistent. Highly experimental!

## SYNTAX

### ByDN
```
Remove-ADDBObject [-Force] -DistinguishedName <String> -DBPath <String> [-LogPath <String>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByGuid
```
Remove-ADDBObject [-Force] -ObjectGuid <Guid> -DBPath <String> [-LogPath <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```
